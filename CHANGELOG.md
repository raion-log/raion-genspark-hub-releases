# RAION Genspark Hub - Changelog

## v1.0.1 (2026-03-19~20)

### 사이드패널 탭 전환 추적 수정
- 아이콘 클릭한 탭에서만 사이드패널 표시되도록 `tabs.onActivated` 리스너 추가
- 다른 탭으로 전환 시 해당 탭에서 사이드패널 자동 비활성화
- 사이드패널 탭 닫힘 시 추적 상태 초기화

### 테마
- 보라색 테마 적용 (은색 → 빛나는 보라색)
- 브랜드 텍스트 그라디언트 보라색 전환

### 정렬 수정
- 이미지 패널 레이아웃 genspark-pro에 맞춤
- 숫자 입력(±) 버튼/입력창 크기 통일
- 설정 토글 가운데 정렬

### 프로젝트 데이터 관리
- chrome.storage.sync 기기 간 자동 동기화
- 프로젝트 내보내기/가져오기 (JSON, 병합 방식)
- URL에서 프로젝트 가져오기 (Gist, Cloudflare Worker 등)
- 프로젝트 개별 삭제 / 전체 삭제 기능
- 기본 프로젝트(예시) 삭제 가능 (숨김 처리)
- 2시간 주기 자동 백업
- default-projects.json v2 포맷 (그룹/순서 포함)

### UI 개선
- 프로젝트 헤더 2줄 구조 (1줄: 프로젝트/새그룹/새프로젝트, 2줄: URL/가져오기/내보내기/전체삭제)
- 새 그룹 버튼 노란색 테마, 새 프로젝트 버튼 보라색 테마
- 프로젝트 카드 버튼 순서: 활성화 → 편집 → 삭제
- "프로젝트" 타이틀 보라색 반짝이 애니메이션
- 전체 글씨 크기 상향 (카드명 15px, 설명 13px, 버튼 13px)
- 활성화 버튼 보라색 배경 + 흰 글자
- "< 홈" / "< 프로젝트 목록" 네비게이션 흰색 + 크기 증가
- 그룹 내 프로젝트 루트 중복 표시 버그 수정

### 버그 수정
- 지침서 전송 성공인데 실패로 표시되던 버그 수정 (content-image.js 메시지 가로채기 해결)
- 불필요한 "이해했습니다" 확인 메시지 제거

### 빌드
- dist 빌드 시 console 출력 비활성화 (disableConsoleOutput: true)
- 버전 통일 (v1.0.1)

### 기타
- 자동 업데이트 확인 기능
