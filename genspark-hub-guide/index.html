URL / 가져오기 / 내보내기

🔗URL

- 강사가 미리 만들어 둔 JSON 데이터를 URL로 변환한 링크

- 해당 URL을 복사 → 붙여넣기하면 그룹·프로젝트 자동 가져오기 처리

- 불러온 지침서는 바로 사용 가능하며, 수정·보완 후 활용도 가능

📂가져오기

- 이전에 내보내기로 저장해 둔 .json 파일을 내 컴퓨터에서 선택하여 불러오는 기능

💾내보내기

- 확장프로그램 상단 "내보내기" 클릭

→ 현재 그룹·프로젝트 데이터가 .json 파일로 다운로드

→ 파일명은 구분하기 쉬운 이름으로 변경하여 보관

⚠️

자동 저장 기능이 있긴 하지만, Chrome 확장프로그램의 저장소 용량이 크지 않습니다. 내용을 수정했다면 꼭 내보내기를 해서 따로 파일을 보관해 주세요. 안 그러면 데이터가 날아갈 수 있습니다.

3

'활성화' 버튼

프로젝트 지침서를 채팅에 적용하는 두 가지 방법

A

새 채팅 시작

새 대화 열기 + AI 모델 설정 + 지침서 입력을 한 번에 진행

* 처음 채팅 시작하거나, 새 채팅에서 다른 지침서를 적용하고 싶을 때 사용

B

이어서 진행

현재 진행 중인 채팅에 프로젝트 지침서만 추가 입력

* 모델 변경 없이 지침서 내용만 이어 붙이는 방식

⚠️

처음 채팅을 시작할 때 설정한 AI 모델은 중간에 바꿀 수 없습니다.

예를 들어 처음에 Claude Opus 4.6으로 시작했는데, "이어서 진행"으로 불러온 프로젝트에 Gemini 3.1 Pro가 설정되어 있어도 모델은 바뀌지 않습니다. Claude Opus 4.6 그대로 유지되고, 지침서 내용만 추가됩니다.
